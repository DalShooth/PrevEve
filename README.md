# WAYLAND EVE-O-PREVIEW CLONE

## HOW TO BUILD
### DEPENDENCIES
- cmake
- make
- qt6-base-dev
- libgstreamer1.0-dev
- libgstreamer-plugins-base1.0-dev
- libpipewire-0.3-dev
### BUILD FROM SOURCE
- Download ZIP
- Unzip to your home directory
- `cd ~`
- `cmake PrevEve-master/`
- `make`
- `chmod +x PrevEve`
- `./PrevEve`
